# 创建 API Key

API Key 是你调用 AI 模型的凭证，格式以 `sk-` 开头。下面介绍如何创建和管理你的 API Key。

## 进入令牌管理

1. 登录 [使用站点](https://api.kkkk24juastin.asia)
2. 在控制台中找到并点击「令牌管理」或「API Key 管理」菜单

## 点击创建令牌

在令牌管理页面，点击「创建令牌」或「新建 API Key」按钮。

## 设置名称、额度、模型限制

创建令牌时，你需要设置以下信息：

| 配置项 | 说明 |
|--------|------|
| **名称** | 给这个 Key 起个名字，方便识别用途（如：ChatBox 用、开发测试用等） |
| **额度限制**（可选） | 设置这个 Key 的最大使用额度，防止意外超支 |
| **模型限制**（可选） | 限制这个 Key 可以使用的模型范围 |

::: tip
如果你是新手，建议先创建一个不限额度和模型的 Key，方便测试。后续可以根据需要创建多个 Key 分别用于不同场景。
:::

## 复制并保存 sk- 开头的 Key

创建成功后，页面会显示你的 API Key。你可以随时在令牌管理页面复制已创建的 Key。

::: warning 安全提示
- API Key 以 `sk-` 开头，是一串很长的字符
- 请勿将 API Key 分享给他人或上传到公开仓库
:::

复制好 API Key 后，你就可以在客户端或代码中使用它来调用 AI 模型了。
