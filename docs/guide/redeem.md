# 兑换码充值

购买服务后，你需要通过兑换码将额度充入账号。以下是详细步骤。

## 打开钱包管理

1. 登录 [使用站点](https://api.kkkk24juastin.asia)
2. 在控制台中找到并点击「钱包管理」或「余额管理」菜单

## 找到兑换码充值

在钱包管理页面中，找到「兑换码充值」或「充值」入口。

## 输入购买获得的兑换码

1. 在输入框中粘贴你购买时获得的兑换码
2. 点击「兑换」或「确认充值」按钮

::: warning 注意事项
- 兑换码通常只能使用一次
- 请确保兑换码复制完整，不要有多余的空格
- 兑换码区分大小写，请仔细核对
:::

## 查看余额是否到账

兑换成功后：

1. 页面会提示充值成功
2. 查看页面上显示的**余额**是否已经增加
3. 你也可以在「消费记录」或「账单」中查看充值记录

如果余额没有到账，请参考 [常见问题 - 余额充值不到账怎么办](/guide/faq#余额充值不到账怎么办)。
