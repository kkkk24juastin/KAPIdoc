# 购买与充值

本站支持两种充值方式：**在线充值**和**兑换码充值**。

## 方式一：在线充值（推荐）

在线充值是最便捷的方式，余额即时到账。

### 操作步骤

1. 登录 [使用站点](https://api.kkkk24juastin.asia)
2. 在控制台中找到并点击「钱包管理」或「余额管理」菜单
3. 选择「在线充值」入口
4. 选择你需要充值的金额
5. 完成支付后，余额将即时到账

::: tip 优势
- 即时到账，无需手动操作
- 支持多种支付方式
- 无需记忆或保管兑换码
:::

---

## 方式二：兑换码充值

如果你通过购买获得了兑换码，可以按照以下步骤充值。

### 操作步骤

1. 登录 [使用站点](https://api.kkkk24juastin.asia)
2. 在控制台中找到并点击「钱包管理」或「余额管理」菜单
3. 找到「兑换码充值」或「充值」入口
4. 在输入框中粘贴你购买时获得的兑换码
5. 点击「兑换」或「确认充值」按钮

::: warning 注意事项
- 兑换码通常只能使用一次
- 请确保兑换码复制完整，不要有多余的空格
- 兑换码区分大小写，请仔细核对
:::

---

## 查看余额是否到账

充值成功后：

1. 页面会提示充值成功
2. 查看页面上显示的**余额**是否已经增加
3. 你也可以在「消费记录」或「账单」中查看充值记录

如果余额没有到账，请参考 [常见问题 - 余额充值不到账怎么办](/guide/faq#余额充值不到账怎么办)。
