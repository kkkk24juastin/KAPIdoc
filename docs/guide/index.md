## API Key 是什么

**API Key**（也叫令牌、Token）是你调用 AI 模型 API 时的身份凭证，格式通常以 `sk-` 开头。

简单理解：

- **API Key** = 你的"钥匙"
- **余额** = 你的"油费"
- 调用 AI 模型 = 用钥匙发动汽车，消耗油费

你需要先创建一个 API Key，然后在客户端或代码中填入这个 Key，才能调用 AI 模型。每次调用会根据使用的模型和 Token 数量从余额中扣费。

::: tip
API Key 是敏感信息，请妥善保管，不要泄露给他人。
:::
