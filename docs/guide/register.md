# 注册与登录

## 打开使用站点

在浏览器中访问使用站点：

> [https://api.kkkk24juastin.asia](https://api.kkkk24juastin.asia)

## 注册账号

1. 打开使用站点后，点击页面上的「注册」按钮
2. 填写注册信息：
   - **邮箱地址**：填写你的常用邮箱
   - **密码**：设置一个安全的密码
   - **确认密码**：再次输入密码
   - **验证码**：如果有的话，填写页面显示的验证码
3. 点击「注册」完成账号创建
4. 如果需要邮箱验证，请前往邮箱查收验证邮件并点击链接完成验证

::: tip
建议使用常用邮箱注册，方便后续找回密码。
:::

## 登录账号

1. 访问 [使用站点](https://api.kkkk24juastin.asia)
2. 点击页面上的「登录」按钮
3. 输入你注册时使用的**邮箱**和**密码**
4. 点击「登录」即可进入控制台

登录成功后，你将看到用户控制台，在这里可以管理你的 API Key、查看余额、查看用量等。

## 忘记密码怎么办

如果你忘记了登录密码，可以通过以下步骤重置：

1. 在登录页面点击「忘记密码」链接
2. 输入你注册时使用的邮箱地址
3. 系统会向你的邮箱发送一封重置密码的邮件
4. 打开邮件，点击重置链接
5. 设置新密码并确认
6. 使用新密码登录

::: warning
如果没有收到重置邮件，请检查垃圾邮件文件夹。如果仍然找不到，请联系客服。
:::
